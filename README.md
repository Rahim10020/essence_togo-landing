# Essence Togo Landing Page

A modern landing page for Essence Togo app, built with Next.js and Tailwind CSS.

## Getting Started

1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Open [https://essencetogo-app.vercel.app/](https://essencetogo-app.vercel.app/)

## Features

- Responsive design
- Hero section
- Features showcase
- How it works
- Screenshots
- Download section
- Footer

## Deploy

Deploy on Vercel or any hosting platform.
